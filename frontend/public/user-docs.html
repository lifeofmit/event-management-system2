System Documentation Guide

Learn how to submit events, upload reports, and manage your assigned directory.

Note: The features you see in the system depend on your assigned role (Admin, Dean, or Coordinator).

1. How to Add a New Event

All system users have the ability to submit events. Navigate to the Add Event tab on your sidebar.

  • Mandatory Fields: Event Name, Category, Date, and Objective are required. You cannot select a date in the future.
  • Required Photos: You must upload at least 1 Geo-Location Photo and 1 Event Photo. You can upload up to 30 photos per section simultaneously.
  • Location Detection: Click the 'Auto Detect Location' button to automatically fill your Latitude and Longitude coordinates based on your device's GPS.

2. Uploading the Final Report

You do not need to have the final report document ready when you initially create the event.

  • Locating the Event: Once the event is finished and your report is written, go to the Event Directory.
  • Uploading: Find the event you created, and click the blue Upload Report button in the Actions column.
  • Formats: You can only upload files in .doc or .docx format.
  • Completion: Once successfully uploaded, the button will change to a purple 'Report' button, allowing anyone with access to download it.

* Security Rule: You will only see the 'Upload Report' button on events that YOU explicitly created.

3. Directory Visibility Rules

The events you are allowed to view in your Event Directory depend heavily on your system role:

  • Coordinators: You will ONLY see events that you have personally created.
  • Deans: You will see events created by yourself, AND events created by any Coordinator assigned to report to you.
  • Admins: You will see events created by yourself, events from your assigned Deans, and events from the Coordinators reporting to those Deans.

4. Data Exporting (Admin Only)

If you hold the Admin role, you have the authority to export data from the system.

  • Generating Reports: At the top right of your Event Directory, you will see buttons for CSV, EXCEL, and PDF.
  • Targeted Exports: Applying filters (like searching for a specific date range or category) before clicking export will ensure your downloaded file only contains the filtered data currently shown on your screen.
