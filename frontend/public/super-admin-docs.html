System Documentation

Comprehensive guide to system configuration, user management, and hierarchical controls.

1. Understanding Your Role

As the Super Admin, you hold the highest level of access in the system. Your primary responsibility is configuring the system for other users.

While you can view, add, and export all events, your day-to-day focus should be maintaining the structural hierarchy (Assigning Deans to Admins, and Coordinators to Deans) and managing event categories.

2. User Management

You have full control over all system accounts.

  • Creating Users: Navigate to the User Management tab, click "Add New User", fill in the details, and assign their specific role.
  • Editing Credentials: Click the blue edit (pencil) icon next to any user to change their Name, Email, or Role.
  • Resetting Passwords: In the edit menu, type a new password in the override box and save. Leave it blank to keep their current password intact.
  • Your Own Profile: Use the blue banner at the top of the User Directory to change your own Super Admin name, email, or password.

3. Hierarchy & Mapping

The system relies on a strict reporting structure to dictate who can see which events: Admin → Dean → Coordinator.

  • Assigning a Dean to an Admin: Edit a Dean's profile. A dropdown will appear allowing you to select which Admin they report to.
  • Assigning a Coordinator to a Dean: Edit a Coordinator's profile. A dropdown will appear allowing you to select which Dean they report to.
  • Clearing Assignments: Select "None (Clear Assignment)" from the dropdown to remove a user from the hierarchy.

4. Event Categories Management

You control the master list of Event Types (e.g., Workshop, Seminar, Cultural) that appear in the Add Event form dropdown.

  • Adding Categories: Go to Event Categories and click Add. It will immediately be available to all users.
  • Disabling Categories: Toggle the green switch to 'off'. The category won't be deleted from past events, but users will no longer be able to select it for new events.

5. Data & Exports

You have unrestricted global access to all events in the system.

  • Filtering: Use the Search bar at the top of the Event Directory to narrow down results by Date Range, Name, or Category.
  • Exporting: Click the CSV, EXCEL, or PDF buttons. The exported report will strictly respect the filters you currently have applied on the screen.
